PROYECTO FINAL - VISUALIZADOR DE CURVAS PARAMÉTRICAS 3D

FORMA DE ABRIRLO SIN VISUAL STUDIO CODE:

1. Descomprime la carpeta completa.
2. Abre el archivo index.html con Google Chrome, Microsoft Edge o Firefox.
3. No necesitas instalar Visual Studio Code.
4. Sí necesitas conexión a internet, porque GeoGebra se carga desde su librería oficial en línea.

IMPORTANTE:
- No borres ni muevas los archivos de la carpeta.
- El archivo .ggb base está incluido como progeofinal.ggb y también como progeofinal_actualizado.ggb.
- La interfaz web agrega y controla algunos objetos extra desde JavaScript, como resultados numéricos y concavidad.

SI NO ABRE BIEN:
- Usa Google Chrome.
- Asegúrate de tener internet.
- Si el navegador bloquea archivos locales, abre la carpeta con Live Server; esto solo sería un respaldo, no debería ser necesario en la mayoría de casos.
